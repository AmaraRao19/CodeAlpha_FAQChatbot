# AI-Powered University FAQ Chatbot

An intelligent University FAQ Chatbot developed using React.js, Flask, NLP, TF-IDF and Cosine Similarity.

## Technologies Used

- React.js
- Flask
- Python
- NLTK
- SpaCy
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

## Features

- University FAQ Chatbot
- NLP Text Preprocessing
- Intelligent Question Matching
- Similarity Threshold
- Professional Chat UI
- Suggested Questions
- Chat History
- Auto Scroll
- Enter Key Support
